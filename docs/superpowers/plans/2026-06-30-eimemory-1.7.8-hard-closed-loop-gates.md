# eimemory 1.7.8 Hard Closed-Loop Gates Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close the remaining behavior gates for callable skills, operator corrections, research/news answer context, and hard metrics.

**Architecture:** Keep the existing 1.7 graph-first/autonomy modules and add narrow enforcement at promotion, answer-context, and metric aggregation boundaries. The patch does not add a parallel loop; it hardens the existing closed loop so evidence can affect behavior and L5 scoring.

**Tech Stack:** Python runtime, SQLite-backed runtime store, OpenClaw hooks, pytest.
